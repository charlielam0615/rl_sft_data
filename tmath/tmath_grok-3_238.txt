### Problem Restatement
We need to find how many values of \( x \) satisfy \( f(f(x)) = 5 \), where the function \( f \) is defined as:
\[
f(x) =
\begin{cases} 
x^2 - 4 & \text{if } x \geq -4, \\
x + 3 & \text{otherwise}.
\end{cases}
\]
This means we need to compute \( f(f(x)) \), set it equal to 5, and determine the number of distinct solutions for \( x \).

### Step 3
[PLANNING]
Action: Reach Goal (Implementation)
Current goal: Compute \( f(f(x)) = f(y) \) and set it to 5.
Method: Substitute \( y = f(x) \) into \( f(y) \), apply the piecewise function based on whether \( y \geq -4 \), or \( y < -4 \), and solve \( f(y) = 5 \).
[/PLANNING]

[IMP]
We need \( f(y) = 5 \). Using the definition of \( f \):
- If \( y \geq -4 \):
\[
f(y) = y^2 - 4 = 5 \implies y^2 = 9 \implies y = \pm 3
\]
- If \( y < -4 \):
\[
f(y) = y + 3 = 5 \implies y = 5 - 3 = 2
\]
Since \( y = 2 < -4 \) is false, discard \( y = 2 \).
Thus, \( y = 3 \) or \( y = -3 \).
[/IMP]

Solve \( f(x) = 3 \):
- If \( x \geq -4 \):
\[
x^2 - 4 = 3 \implies x^2 = 7 \implies x = \pm \sqrt{7}
\]
Since \( \sqrt{7} \approx 2.645 > -4 \) and \( -\sqrt{7} \approx -2.645 > -4 \), both are valid.
- If \( x < -4 \):
\[
x + 3 = 3 \implies x = 0
\]
Since \( 0 < -4 \) is false, discard \( x = 0 \).

Solutions: \( x = \sqrt{7}, -\sqrt{7} \).

**Case 2: \( y = -3 \)**  
Solve \( f(x) = -3 \):
- If \( x \geq -4 \):
\[
x^2 - 4 = -3 \implies x^2 = 1 \implies x = \pm 1
\]
Since \( 1 > -4 \) and \( -1 > -4 \), both are valid.
- If \( x < -4 \):
\[
x + 3 = -3 \implies x = -6

Since \( -6 < -4 \), this is valid.
